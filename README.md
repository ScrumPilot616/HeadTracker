# HeadTracker


## Features
Output selectable between
- PPM
- Serial
- WiFi


## Hardware

### Required
- WEMOS D1 MINI / WEMOS D1 MINI PRO
- MPU9250

### Optional
- Reset button
- Voltage regulator e.g. 7805
- 3.5mm jack 4 Pin
